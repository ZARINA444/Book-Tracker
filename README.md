# Book Tracker

**Автор:** Зарина Загидуллина

## Описание
Приложение для учёта книг с фильтрацией и JSON.

## Запуск
python book_tracker.py

## Примеры
1. Добавление книги
2. Фильтр по жанру
3. Фильтр по страницам (>200)
4. Комбинированный фильтр
5. Сохранение в JSON
