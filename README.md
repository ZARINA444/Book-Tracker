# Book Tracker

**Автор:** Зарина Загидуллина

## Описание
Позволяет добавлять книги (название, автор, жанр, количество страниц), удалять их, фильтровать по жанру и страницам (>200), а также сохранять/загружать коллекцию в JSON-файл.

## Запуск
1. Убедитесь, что установлен Python 3.6+.
2. Скачайте файл `book_tracker.py`.
3. Запустите файл.

## Примеры
1. Добавление книги
2. Фильтр по жанру
3. Фильтр по страницам (>200)
4. Комбинированный фильтр
5. Сохранение в JSON
